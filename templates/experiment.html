BLASTing the Flu Virus

Experiment based on the procedure at Science Buddies

Background:

Influenza is a virus that affects the upper respiratory tract. Common in cold and dry weather, the virus has a 'season'. There are 3 types of the virus: type A, type B, and type C. Type A can spread quickly and in many mammals while type B and C only affect humans. Type C does no real damage as it causes a mild infection, while type B can be devastating. Type A and B both have sub-categories based on the two main proteins of the virus: hemagglutinin and neuraminidase (H and N). The different evolution on the viruses rely on changes made to these proteins. Hemagglutinin are used in attaching the virus onto a cell, while the neuraminidase is used to spread the infection. Flu vaccines contain three different strands of these H and N proteins usually and each one helps prevent the flu from these strands. Scientists use a technique called sequence alignment to try to get the best match possible for the yearly vaccine. BLAST (Basic Local Alignment Search Tool) is an online based bioinformatics tool for aligning sequences.
